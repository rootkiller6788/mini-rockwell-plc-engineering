### RWTH Aachen: Industrial Control Systems

- Topic: Industrial communication protocols
- Alignment: EtherNet/IP & CIP protocol stack implementation

- Topic: Deterministic network behavior
- Alignment: RPI/API calculation, connection timeout computation

- Topic: Industrial network protocol stacks
- Alignment: Message Router, EPATH encoding, object-oriented addressing

### Berkeley ME233: Advanced Control
